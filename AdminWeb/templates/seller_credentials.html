API Secret
Hướng dẫn sử dụng nhanh

Seller sử dụng api_key và api_secret để ký HMAC-SHA256 mọi request tới API.

Xem chi tiết tại API Documentation.

Code mẫu curl:
curl -H "X-Seller-Key: {{ key.api_key }}" \
     -H "X-Timestamp: $(date +%s)" \
     -H "X-Nonce: $(openssl rand -hex 16)" \
     -H "X-Signature: <hmac_signature>" \
     https://your-api-domain.com/seller/v1/products
Code mẫu Python:
import hmac, hashlib, time, secrets, requests

api_key = "{{ key.api_key }}"
api_secret = "{{ api_secret }}"

method = "GET"
path = "/seller/v1/products"
timestamp = str(int(time.time()))
nonce = secrets.token_hex(16)
body = ""
body_hash = hashlib.sha256(body.encode()).hexdigest()
message = method + path + timestamp + nonce + body_hash
signature = hmac.new(
    api_secret.encode(), message.encode(), hashlib.sha256
).hexdigest()

resp = requests.get(
    "https://your-api-domain.com" + path,
    headers={
        "X-Seller-Key": api_key,
        "X-Timestamp": timestamp,
        "X-Nonce": nonce,
        "X-Signature": signature,
    },
)
print(resp.json())
